# Smart Expense Tracker

## Version

1.0.0

---

# Project Vision

To build a production-ready Smart Expense Tracker that helps individuals and groups manage their finances, track expenses, split shared costs, and visualize spending through an intuitive and secure web application.

---

# Problem Statement

Managing personal and shared expenses manually is difficult.

Common challenges include:

- Forgetting where money was spent
- Exceeding monthly budgets
- Losing receipts
- Difficulty splitting group expenses
- Confusion during settlements
- Lack of spending insights

This project aims to solve these problems through a centralized web application.

---

# Project Objectives

- Track income
- Track expenses
- Manage monthly budgets
- Upload receipts
- Split expenses among friends
- Simplify settlements
- Generate reports
- Visualize spending trends
