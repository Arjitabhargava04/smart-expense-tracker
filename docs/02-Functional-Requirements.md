# Functional Requirements

Version: 1.0

---

# Module 1 - Authentication

## FR-1

The system shall allow a user to register using:

- Full Name
- Email
- Password

---

## FR-2

The system shall allow users to login using email and password.

---

## FR-3

The system shall allow users to logout.

---

## FR-4

The system shall allow users to reset forgotten passwords.

---

## FR-5

The system shall support Google Authentication.

---

# Module 2 - User Profile

## FR-6

The user shall be able to update profile information.

---

## FR-7

The user shall be able to upload a profile picture.

---

# Module 3 - Expense Management

## FR-8

The user shall be able to add an expense.

---

## FR-9

The user shall be able to edit an expense.

---

## FR-10

The user shall be able to delete an expense.

---

## FR-11

The user shall be able to categorize expenses.

---

## FR-12

The user shall be able to search expenses.

---

## FR-13

The user shall be able to filter expenses.

---

# Module 4 - Income

## FR-14

The user shall be able to add income.

---

## FR-15

The user shall be able to edit income.

---

## FR-16

The user shall be able to delete income.

---

# Module 5 - Budget

## FR-17

The user shall be able to create monthly budgets.

---

## FR-18

The system shall notify users when the budget limit is exceeded.

---

# Module 6 - Dashboard

## FR-19

The dashboard shall display:

- Total Income
- Total Expenses
- Remaining Budget
- Monthly Summary

---

## FR-20

The dashboard shall display spending charts.

---

# Module 7 - Groups

## FR-21

The user shall be able to create groups.

---

## FR-22

The user shall invite members.

---

## FR-23

The user shall add shared expenses.

---

## FR-24

The system shall support:

- Equal Split
- Exact Split
- Percentage Split
- Share Split

---

## FR-25

The system shall calculate balances automatically.

---

# Module 8 - Settlement

## FR-26

The system shall simplify debts.

---

## FR-27

The user shall settle outstanding balances.

---

# Module 9 - Reports

## FR-28

The user shall export reports as PDF.

---

## FR-29

The user shall export reports as CSV.

---

# Module 10 - Notifications

## FR-30

The system shall send budget reminders.

---

## FR-31

The system shall send settlement reminders.
